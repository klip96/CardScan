==========================================================
  СКАНЕР ВИЗИТОК — как собрать установщик на Windows
==========================================================

ЧТО ЭТО
  Исходники приложения + скрипт, который сам соберёт .exe и установщик.
  Готового .exe в архиве нет — его собирает сам Windows одним действием.

НИЧЕГО ЗАРАНЕЕ СТАВИТЬ НЕ НУЖНО.
  Скрипт сборки сам проверит наличие Python и Inno Setup, и если их нет —
  установит автоматически (через winget, а если его нет — скачает официальные
  установщики и поставит тихо). Нужен только ИНТЕРНЕТ на время сборки.

КАК СОБРАТЬ (одно действие):
  1) Распакуйте этот архив в любую папку.
  2) Зайдите в папку installer и дважды кликните по:  build_all.bat
     - Если чего-то не хватает — появится запрос прав администратора (UAC),
       подтвердите его (нужен только для установки Python / Inno Setup).
     Дальше скрипт сам:
        - установит Python и Inno Setup, если их нет,
        - создаст окружение и поставит зависимости,
        - соберёт CardScan.exe (PyInstaller),
        - скомпилирует установщик (Inno Setup).
  3) Готовый установщик появится в:  installer\Output\
     Его и раздавайте на рабочие ПК — там ставится двойным кликом.

ЕСЛИ АВТО-УСТАНОВКА НЕ СРАБОТАЛА (нет интернета/ограничения):
  Поставьте вручную и повторите build_all.bat:
    - Python 3.11+ — https://www.python.org/downloads/ (галочка "Add Python to PATH")
    - Inno Setup 6 — https://jrsoftware.org/isdl.php

ПОСЛЕ УСТАНОВКИ НА РАБОЧЕМ ПК:
  - Запустите "Сканер визиток" — откроется страница в браузере.
  - Нажмите "Настройки" и подключите Google-таблицу
    (загрузите ключ, вставьте ссылку, "Проверить подключение").
  - Для распознавания: впишите API-ключ облака (Gemini/OpenAI/Claude)
    ЛИБО выберите локальный режим (см. README).
  - Телефон: в той же Wi-Fi откройте http://cardscan.local:8000
    или отсканируйте QR на странице "Подключить телефон".

ЛОКАЛЬНАЯ НЕЙРОНКА (необязательно, офлайн):
  При установке будет предложено поставить Ollama и скачать модель
  (нужен интернет один раз ~1-2 ГБ). На ПК без видеокарты работает
  медленнее облака — по умолчанию рекомендуется облако.

Подробности — в README.md.
